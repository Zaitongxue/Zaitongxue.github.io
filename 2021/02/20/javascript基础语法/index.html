变量与常量

变量

在javascript中，变量是指一个可以被改变的量。变量的值在程序运行过程中是可以改变的。

变量的命名

原则：
• 变量由字母，下划线，$或数字组成，并且第一个字母必须是“字母，下划线或$”。
• 变量不能是系统关键字和保留字
至于关键字，保留字等在书225页可查。
注：变量的命名一定要区分大小写。同一个字母大写和小写就是两个不同的变量。

变量的使用

• 变量的声明
• 变量的赋值
所有javascript变量都由var声明。
语法：var 变量名 = 值;
说明：var为声明符，=为赋值符。

在这个例子中，我们使用var来定义一个变量，变量名为a，值为10.然后使用document.write()方法输出这个变量的值。对于变量的命名，我们尽量取一些有意义的英文名或英文缩写。
一个var也可以同时声明多个变量名，其中变量名之间必须用英文逗号隔开，如下:
var a=10,b=20,c=30;
以上代码等同于：
var a=10;
var b=20;
var c=30;

常量

在javascript中，常量指的是一个不能改变的量。我们可以把常量看成一种特殊的变量，之所以特殊，是因为它的值是不会变的。一般情况下，常量名全部大写。常量用const声明。
如下所示：
const DEBUG=1;
程序是会变化的，因此变量比常量有用得多。

数据类型

在javascript中，数据类型有两种：基本数据类型和引用数据类型。其中，基本数据类型只有一个值，而引用数据类型可以含有多个值。
在JavaScript中，基本数据类型有五种：数字，字符串，布尔值，未定义值和空值。而常见的引用数据类型有两种：数组，对象。
在JavaScript中，常见的数据类型有6种：
（1）数字；
（2）字符串；
（3）布尔值；
（4）转义字符；
（5）未定义值；
（6）空值；

数字

数字是最基本的数据类型。在javascript中，数字是不分整形（int）和浮点型（float）的。记住一句话就可以了：在javascript中，所有变量都是用var声明。

字符串

字符串中名字上很容易理解，就是一串字符。在JavaScript中，字符串都是用英文单引号或者双引号括起来的。
注：单引号括起来的字符串中，不能有单引号，只能含双引号。同理，双引号也是一样。如果不这样规定，javascript无法分辨哪两个单引号（双引号）是一对的。

布尔值

布尔值只有两个，即true（真），false（假）。布尔值最大的用处就是：选择结构的条件判断（if语句等）。

未定义值

未定义值指的是如果一个变量已经用var声明了，但是并没有对这个变量进行赋值，此时该变量的值就是“未定义值”。其中，未定义值用undefined表示。

空值

在javascript中，空值用null表示。如果一个变量的值等于null，如var n=null;，则表示系统没有给这个变量n分配内存空间。null和undefined非常相似，但是也有一定区别。

运算符

在javascript中，运算符指的是变量或者值进行运算操作的符号。其中，常见的运算符有以下五种：
（1）算术运算符；
（2）赋值运算符；
（3）比较运算符；
（4）逻辑运算符；
（5）条件运算符；

对于算数运算符，我们需要重点掌握这三种：加法运算符，自增运算符，自减运算符。

加法运算符

我们需要注意以下三点：
• 数字+数字=数字
• 字符串+字符串=字符串
• 字符串+数字=字符串
注：如果想往字符串里塞点东西，就用加号链接，用英文逗号隔开。

自增运算符

++是自增运算符，表示在原来的值的基础上再加上1.i++等价于i=i+1；自增运算符有以下两种情况：
• i++
i++指的是在使用i之后，再让i的值加上i
i=1;
j=i++;
以上这段代码等同于：
i=1;
j=i;
i=i+1;
因此，以上的执行结果为i=2，j=1。
• ++i
++i指的是在使用i之前，先让i的值加上i：
i=1;
j=++i;
以上这段代码等同于：
i=1;
i=i+1;
j=i;
因此，上面执行的结果是i=2，j=2。
i=i+1的使用位置，是根据++的使用位置来决定的。

自减运算符

—是自减运算符，表示在原来的值的基础上再减去1.i—等价于i=i-1；自减运算符同样也有和自增运算符一样的两种情况，这里不多追叙了。

与运算用&&表示。如果&&两边的值都为true，则结果返回true，如果有一个为false或者两个都为false，则结果返回false。

2.“或”运算

或运算用||表示。如果||两边的值都为fasle，则返回false；如果有一个为true或者两个为true，则结果返回true。

3.“非”运算

非运算用英文感叹号表示。非运算操作的对象只有一个。当!右边的值为true时，最终结果为false;当!右边的值为false时，最终结果为true。

条件运算符

除了以上这些常用的运算符之外，javascript还为我们提供了一种特殊的运算符：条件运算符。条件运算符也叫三目运算符。用英文问号表示。
语法：var a = 条件 ? 表达式1 : 表达式2;
说明：当条件为true，我们选择的是“表达式1”，也就是var a=表达式1;当条件为false时，我们选择的是“表达式2”，也就是var a=表达式2。
注：a只是一个变量名，可更改。

表达式与语句

一个表达式包含“操作数”和“操作符”两部分。操作数可以是变量，也可以是常量。操作符就是我们之前学的运算符。每一个表达式都会产生一个值。
语句，简单来说就是用英文分号分开的一句代码。一般情况下，一个分号对应一个语句。

1+2是一个表达式，而整一句代码“var a=1+2;”就是一个语句。

类型转换

类型转换，指的是将一种数据类型转换为另外一种数据类型。“javascript会自动将数字转换为字符串”指的就是类型转换。
在javascript中，共有两种类型转换：
• 隐式类型转换
• 显式类型转换
隐式类型转换指的是javascript自动进行的类型转换。显式类型转换，指的是需要我们手动用代码强制进行的类型转换。
